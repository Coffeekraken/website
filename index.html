Who are Coffeekraken

We are a young collectve of front-end creative developers with a goal in mind. Build tools to make every team working day life better. This is our first and only concern. All our tools are build around that purpose.
All what we provide are some cool tools that you can use the way you want. These tools features cover a large scope of the front-end workflow (styleguide generation, colors/fonts management, etc...). You can use only the parts that you need and let the rest aside...

Discover more on Coffeekraken and how to join us

Our tools

Carpenter

Styleguide/Documentation

Nice web interface that display all your website styleguide by parsing directly your CSS codebase as well as aggregating all the documentations needed by your developers team to work as one.

Check it out!

Sugar

SASS & JS toolkit

Powerful toolkit that gives you access to a lot of SASS mixins, functions, colors and fonts management as well as a lot of javascript features like DOM helpers, classes, utils and webcomponents.

Check it out!

Gridle

SASS grid system

Fully customizable SASS grid system that let you create from simple 12 columns layouts, to any columns numbers complexe ones with ease. Support both default (float) and flexbox based systems.

Check it out!

HTML Boilerplate

Clean and fully featured (webpack, babel, node-sass) boilerplate

Check it out!

Docblock to markdown

Parse and convert docblocks from codebase to markdown

Check it out!

Testing stack

Integrate quicly javascript tests inside your project

Check it out!

Webcomponents

Clean and growing library of webcomponents.

Check it out!

We strongly believe that the way to success is by helping each others. That’s why our primary target for our tools is to be published as fully open source projects that you can (and we hope you will) modify, enhance and share!
